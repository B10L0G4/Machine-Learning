# PCA 

> - Análise de componentes pricipais ou Principal Components Analysis(PCA) é um conjunto de técnicas de redução de dimensionalidade que são usadas para reduzir o numero de variaveis em um conjunto de dados, mantendo a maior parte da informação ou seja a variância. (variância é a medida de dispersão dos dados em torno da média)
>
> ### Como funciona o PCA?
>
> - O PCA funciona da seguinte forma:
>
> 1. Escolhe um numero de componentes principais
> 2. Calcula a variância de cada componente
> 3. Calcula a variância acumulada
> 4. Escolhe o numero de componentes principais que explicam a maior parte da variância
>
> A projeção dos dados é feita por meio de uma transformação linear (tranasformação linear é uma transformação que preserva a distância entre os pontos) qu e mapeia os dados de uma espaço de dimensão n para um espaço de dimensão k, onde k é menor que n (n é o numero de variaveis do conjunto de dados) e k é o numero de componentes principais.)
>
> Aplicação 
>
> O PCA é usado para reduzir a dimensionalidade de um conjunto de dados, para que possa ser visualizado em um grafico 2D ou 3D.
